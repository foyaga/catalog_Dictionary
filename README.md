# catalog_Dictionary
#### 脚本作用：对jsfinder获取到的网站url拆分成目录，对目录进行排序形成字典。
---
## 使用教程
#### 把要爬取的URL放到url.txt里面，然后python运行catalog.py文件就行。

#### 目录排序结果会直接在res.txt中进行更新。
